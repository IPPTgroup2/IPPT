Privacy Policy for IPPT

This Privacy Policy outlines how user data is collected, used, and protected by the IPPT System developed by BSIT student researchers.

2. Information Collected


The IPPT System collects the following user data:

- First name, Middle name, Last name
- Email
- Faculty Number
- Course
- Passwords
- Attendance records
- Service requests

3. Use of Information


User data is used for the following purposes:

- QR code attendance tracking
- Service request processing and notifications
- Generating reports for attendance records and service requests

4. Data Security


User data is handled in accordance with R.A. 10173, also known as the Data Privacy Act, to ensure its protection.

5. User Rights


Users have the right to:

- Access their personal data
- Correct inaccuracies in their personal data
- Delete their personal data
- Withdraw consent for data processing

6. Changes to Privacy Policy


The developers may update or modify the Privacy Policy at their discretion. Users will be notified of significant changes.
